👉 TO EDIT YOUR CONTENT (Super Easy!):

   1. Open "config.js" with Notepad (or any text editor)
   2. Change text between quotation marks " "
   3. Save (Ctrl+S or Cmd+S)
   4. Refresh browser (F5)
   5. See your changes instantly!

🆘 SOMETHING BROKE?

   1. Undo changes (Ctrl+Z)
   2. Open config.BACKUP.js
   3. Copy everything
   4. Paste into config.js
   5. Save and refresh
   6. You're back to working version!
