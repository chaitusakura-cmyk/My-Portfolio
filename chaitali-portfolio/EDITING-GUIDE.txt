💡 EDITING EXAMPLES:

EXAMPLE 1: Change Your Name
───────────────────────────────────────────────────────────
BEFORE:  personal: { name: "Chaitali Avadhani" },
AFTER:   personal: { name: "John Smith" },

EXAMPLE 2: Update Hero Title
───────────────────────────────────────────────────────────
BEFORE:  title: "Crafting Words That Convert & Connect",
AFTER:   title: "Your Custom Title Here",

EXAMPLE 3: Change Statistics
───────────────────────────────────────────────────────────
BEFORE:  { number: "5+", label: "Years Experience" },
AFTER:   { number: "10+", label: "Years Experience" },

➕ HOW TO ADD NEW ITEMS:

ADD A PORTFOLIO ITEM:
───────────────────────────────────────────────────────────
1. Find: portfolio: [
2. Copy an existing item (everything from { to },)
3. Paste after the last item
4. Add comma after previous item
5. Update your new item's content

Example:
{
  icon: "📝",
  category: "Blog",
  title: "Your Article Title",
  description: "Brief description here.",
  link: "https://your-link.com"
},

ADD AN EXPERTISE AREA:
───────────────────────────────────────────────────────────
1. Find: expertise: [
2. Copy an existing expertise block
3. Paste and update

Example:
{
  icon: "🎯",
  title: "Your Expertise",
  description: "Description here.",
  points: [
    "Point 1",
    "Point 2",
    "Point 3"
  ]
},

ADD A SKILL:
───────────────────────────────────────────────────────────
{ icon: "✍️", name: "Your Skill" },

ADD A TESTIMONIAL:
───────────────────────────────────────────────────────────
{
  text: "Testimonial quote here.",
  author: "Client Name",
  position: "Their Title, Company"
},

⚠️ IMPORTANT RULES:

✓ Only edit text between " " quotation marks
✓ Keep ALL commas ,
✓ Keep ALL brackets { } and [ ]
✓ Keep ALL colons :
✓ DON'T delete any structural elements

❌ BAD:  name: John Smith
✓ GOOD: name: "John Smith"

❌ BAD:  { number: "5+" label: "Years" }
✓ GOOD: { number: "5+", label: "Years" }

🔧 TROUBLESHOOTING:

PROBLEM: Content not showing
SOLUTION: Save config.js and refresh browser (Ctrl+F5)

PROBLEM: Page looks broken
SOLUTION: Check for missing commas or quotes in config.js

PROBLEM: Made a mistake
SOLUTION: 
  1. Open config.BACKUP.js
  2. Copy everything
  3. Paste into config.js
  4. Save and refresh

PROBLEM: Changes not appearing
SOLUTION: Hard refresh browser
  • Windows: Ctrl + F5
  • Mac: Cmd + Shift + R
